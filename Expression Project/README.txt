This program will either return a solution or return "0" for false or "1" for true depending on what the expression is asking for.

User Guide:
•	Put all files in same folder except for the word doc.
•	Put the expression(s) you want evaluated in the text file: “expressionText.txt”
  o	Only one expression per line
•	Run module3.cpp
•	The output will be evaluated expression.
  o	For Boolean questions.  0 is false, and 1 is true.
